\begin{align*} m=65\text{ mph is positive}\amp\amp\amp\text{Recall slope is } \dfrac{\text{rise}}{\text{run}}=\frac{\text{change in } D\text{ miles}}{\text{change in } t\text{ hour}} \end{align*}

  • For an additional \(2\) hours driving using cruise, the model predicts you will travel an additional \((65)(2) = 130\) miles.\(\checkmark\)
