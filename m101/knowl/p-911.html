First you need some definitions.

Experiment

An activity that has specific results that can occur, but it is unknown which results will occur.

Outcomes

The results of an experiment.

Event

A set \(A\) of certain outcomes of an experiment that you want to have happen.

Sample Space

collection of all possible outcomes of the experiment. Usually denoted as \(SS\text{.}\)

Event space

The set of outcomes that make up an event. The symbol is usually a capital letter.
